# Solarization Planner (NiceGUI)

Browser-first solar sizing tool: PV + battery + pump + carbon, ready for **Codespaces → Render**.

## Quick start (Codespaces)
1. Click **Code → Create codespace on main**.
2. Wait for the container to finish installing dependencies.
3. Run locally:
   ```bash
   source .venv/bin/activate
   python main.py
